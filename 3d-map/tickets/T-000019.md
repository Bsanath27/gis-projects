# T-000019: Advanced Pincode Search UI

## Description
Implement an advanced search interface for pincodes with autocomplete, filtering, and map integration. Enhance the basic text input with a modern search experience.

## Requirements
1. **Autocomplete Search**
   - As user types, show matching pincodes and office names
   - Display in dropdown list with pincode + office name
   - Click to select and auto-highlight on map

2. **Search Filters**
   - Filter by district (checkbox or dropdown)
   - Filter by office type (S.O, H.O)
   - Combined filters (e.g., "district: Kanniyakumari + type: S.O")

3. **Search Results Display**
   - Show result count
   - Display matching pincode details (pincode, office name, district, type)
   - Click result → highlight on map and fly to location

4. **Recent Searches**
   - Store last 5 searches in localStorage
   - Quick access to recently searched pincodes

5. **Keyboard Navigation**
   - Arrow keys to navigate dropdown results
   - Enter to select
   - Escape to close dropdown

## UI Design
- Single search bar with dropdown
- Category buttons for quick filter (All, S.O only, H.O only)
- Results showing pincode badges + office name
- Monochrome design (consistent with T-000016)

## Implementation Notes
- Build a search index from GeoJSON on load
- Use efficient string matching (includes or fuzzy search)
- Cache search results for performance
- Add debounce to search function (300ms)

## Acceptance Criteria
- [ ] Autocomplete dropdown functional
- [ ] Filters working correctly
- [ ] Results display with correct information
- [ ] Map highlights when result selected
- [ ] Keyboard navigation works
- [ ] Recent searches stored and displayed
- [ ] Performance: search completes <100ms
